# Просмотр источника (Frontend)

## Цель (Value)
- **Для пользователя:** Обеспечить прозрачность и проверяемость информации, предоставляя прямые ссылки на источники для каждого блока контента.

## Требования (Requirements)
1.  **UI для источника:** Отображать иконку источника (например, в виде скрепки или глобуса) рядом с каждым блоком или альтернативой, у которых есть источник.
2.  **Ссылка на источник:** Иконка должна быть кликабельной ссылкой, которая открывает URL-источник в новой вкладке.
3.  **Данные из манифеста:** Ссылка на источник должна браться из метаданных соответствующего блока в `manifest.json`.

## Критерии приёмки (Acceptance Criteria)
- Иконка источника корректно отображается для всех блоков, у которых он есть.
- Клик по иконке открывает правильную ссылку в новой вкладке.
- Написаны тесты, проверяющие данную функциональность.

## Ограничения (Constraints)
- Эта задача является чисто frontend-задачей.
- Проверка валидности и доступности ссылок не входит в скоуп данной задачи.
