# Операционное управление AI-задачами (Frontend)

## Цель (Value)
- **Для администраторов/модераторов:** Предоставить инструменты для мониторинга и управления жизненным циклом фоновых AI-задач.

## Требования (Requirements)
1.  **Панель мониторинга:** Интегрировать готовую или разработать собственную панель для мониторинга очередей задач в административный интерфейс.
2.  **UI для управления:** Создать UI для взаимодействия с API по управлению задачами (перезапуск, отмена).

## Критерии приёмки (Acceptance Criteria)
- Административная панель корректно отображает актуальное состояние очередей и задач.
- Администратор может успешно перезапустить или отменить задачу через UI.
- Написаны тесты, проверяющие frontend-часть функционала.

## Ограничения (Constraints)
- В первой итерации предпочтение отдается интеграции готового решения для дашборда.
